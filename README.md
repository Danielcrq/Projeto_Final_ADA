# Projeto_Final_ADA
MovieFlix plataforma simples de cadastro e avaliação de filmes
O projeto MovieFlix cria uma pipeline com extração de dados do site OMDB de filmes armazenando arquivos CSV com dados brutos (movies.csv)(users.csv)(ratings.csv), depois faz a build da imagem Docker e seu Push.

O Dockerfile é responsável por gerar um Docker para colocar o projeto dentro dele.
